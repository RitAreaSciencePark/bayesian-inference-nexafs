# Bayesian NEXAFS Analysis Pipeline
 
A machine-learning pipeline for the automated, uncertainty-aware analysis of Near-Edge X-ray Absorption Fine Structure (NEXAFS) spectra. Built on a Bayesian framework, it extracts physical parameters from spectroscopic data together with rigorous uncertainty estimates, going beyond conventional peak-fitting. The pipeline is designed for multiplexing data — multiple spectral edges measured on the same sample across varying environments — and runs on HPC for fully automated, end-to-end analysis. It has been applied to operando NEXAFS measurements of a TiO₂ sample at the APE-HE beamline of the Elettra synchrotron, jointly analysing the Ti L₂,₃ and La M edges across a vacuum reference and seven gas environments.
