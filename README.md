# Bayesian NEXAFS Analysis Pipeline

An automated pipeline for the quantitative, uncertainty-aware analysis of Near-Edge X-ray Absorption Fine Structure (NEXAFS) spectra. Unlike conventional peak-fitting tools that return a single best-fit set of parameters, it reconstructs the full posterior probability distribution over all spectral parameters, yielding rigorous uncertainty estimates and inter-parameter correlations. The workflow combines massively parallel stochastic global optimisation — used to locate the distinct minima of the χ² landscape — with gradient-based Bayesian inference via Hamiltonian Monte Carlo (HMC), and wraps these in automated diagnostics (convergence statistics, corner and trace plots, and model decompositions). Its five stages (global optimisation → candidate ranking → posterior sampling → diagnosis → best-result selection) share a single JSON configuration and a common forward model, and are orchestrated on HPC through SLURM so that an entire dataset can be analysed end-to-end from a single command.

Each spectrum is modelled as the sum of a smooth polynomial background, Skew-Voigt absorption peaks, and arctangent edge steps — a flexible lineshape parametrisation that captures both symmetric and asymmetric features. The pipeline is designed for multiplexing data, where several spectral edges are measured on the same sample across many environments. It has been applied to operando NEXAFS measurements of a TiO₂ sample at the APE-HE beamline of the Elettra synchrotron, jointly analysing the Ti L₂,₃ edge (455–475 eV, probing the crystal-field-split Ti 3d empty states) and the La M edge (probing La 3d → 4f transitions) across a vacuum reference and seven gas environments (He, H₂, CO, and O₂ mixtures).
