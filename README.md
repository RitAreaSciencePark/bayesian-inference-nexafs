# bayesian-inference-nexafs
Automated Bayesian pipeline for NEXAFS spectroscopy: instead of single best-fit values, it recovers full posterior distributions over spectral parameters via parallel global optimisation and Hamiltonian Monte Carlo, with automated HPC diagnostics. Applied to multiplexing Ti L₂,₃ and La M edge operando data from APE-HE line at ELETTRA Syncrothron.
